# Bending-Wave-Modeling
This project models bending waves in Saturn's rings using a windowed least squares method and a convolutional neural network. The data is from the Cassini ISS camera on revolutions 114 and 115. I/F is extracted using a normalized radial profile, and wavelet transform.
